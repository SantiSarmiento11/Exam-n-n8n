Configuración rápida

Edita js/config.js y reemplaza N8N_WEBHOOK_URL por la URL real del webhook. Para Google Maps, agrega tu API key al script del final de esta página.

  • El envío usa FormData para transportar texto, coordenadas e imagen.
  • El campo reportId se genera en el navegador para trazabilidad inicial.
  • n8n puede responder JSON con caseId para mostrarlo al ciudadano.

La información enviada debe corresponder a hechos verificables y de interés público.
